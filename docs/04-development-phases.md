# Development Phases

## Phase 1 — System Design and Architecture
Deliverables:
```
- System architecture document
- Hardware block diagram
- API design
- Database schema
- Communication payload definition
- Dashboard wireframe
```
Estimated duration: 2 to 3 weeks

---

## Phase 2 — Cloud Platform and API Prototype
Deliverables:
```
- FastAPI backend
- PostgreSQL database
- InfluxDB or TimescaleDB integration
- MQTT broker
- OpenAPI documentation
- Simulated sensor data ingestion
- Basic REST API
- WebSocket event stream
```
Estimated duration: 3 to 5 weeks

---

## Phase 3 — Dashboard and 3D Visualization
Deliverables:
```
- Web dashboard
- Sensor map
- Device status panel
- Road condition charts
- Alert list
- 3D road visualization
- Real-time vehicle event display
```
Estimated duration: 4 to 6 weeks

---

## Phase 4 — Smart Reflector Hardware Prototype
Deliverables:
```
- Electronics design
- Sensor integration
- Power system prototype
- LoRaWAN firmware
- Battery monitoring
- Solar charging test
- Vibration event detection firmware
- Enclosure prototype
```
Estimated duration: 6 to 10 weeks

---

## Phase 5 — Gateway and LoRaWAN Integration
Deliverables:
```
- Outdoor gateway configuration
- ChirpStack setup
- Device registration
- LoRaWAN payload decoder
- MQTT integration
- End-to-end telemetry test
```
Estimated duration: 2 to 4 weeks

---

## Phase 6 — Controlled Road Test
Deliverables:
```
- Installation of 6 smart reflector nodes
- Controlled vehicle pass tests
- Sensor calibration

- Speed and size estimation validation
- Battery and solar performance report
```
Estimated duration: 4 to 6 weeks
