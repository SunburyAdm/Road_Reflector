# Tools

Helper scripts for prototype development and sensor data analysis.

## Live serial plotter

```
python plot_sensors_live.py --port COM6 --baud 115200
```

Plots live sensor data streamed from the ESP32-C3 bench prototype.

> Prototype firmware: [`../01-device-layer/prototype-esp32c3/`](../01-device-layer/prototype-esp32c3/).
